# ZIGZAG_FIBO Dashboard — XAUUSD

Dashboard Android trading XAUUSD: harga live WebSocket, sinyal ZigZag+Fibonacci, RSI-14 per timeframe.

## Build APK via GitHub Actions (tanpa expo.dev)

1. Push repo ini ke GitHub
2. Push ke branch `main` → Actions otomatis berjalan
3. Download APK: tab **Actions** → run terbaru → **Artifacts** → `ZIGZAG_FIBO-APK`

> Tidak perlu akun expo.dev atau token apapun.

## Dev lokal

```bash
npm install
npm start   # scan QR dengan Expo Go di Android
```
